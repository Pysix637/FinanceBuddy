# FinanceBuddy — производственная практика WPF

Тема проекта: персональный финансовый дневник для учёта доходов и расходов.

## Что реализовано
- WPF приложение на .NET 8 (MVVM).
- Главное окно:
  - список операций (DataGrid);
  - добавление операции;
  - удаление выбранной операции;
  - редактирование выбранной операции (отдельное окно);
  - управление категориями (отдельное окно);
  - сводка: доход / расход / баланс.
- Сервисы:
  - парсер суммы (ru-RU + invariant);
  - сервис расчёта итогов;
  - сервис открытия окон редактирования.
- Слой Data:
  - EF Core + SQLite (локальный файл базы в AppData\Local\FinanceBuddy\financebuddy.db);
  - тестовые данные (seed) при первом запуске.
- Документы и диаграммы в папке `docs/` + изображения в `docs/img/`.

## Как запустить
1. Открой `FinanceBuddy.sln` в Visual Studio 2022 (Windows).
2. Убедись, что установлен workload **.NET Desktop Development**.
3. Visual Studio предложит восстановить NuGet-пакеты — согласись (Restore).
4. Запусти проект **FinanceBuddy** (F5).

## Примечания
- Если нужно «с нуля» проверить тестовые данные, удали файл базы:
  `%LOCALAPPDATA%\FinanceBuddy\financebuddy.db`
