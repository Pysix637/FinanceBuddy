# FinanceBuddy — производственная практика (WPF)

Тема проекта: **персональный финансовый дневник** для учёта доходов и расходов.

## Что я уже сделал
- создал WPF‑приложение на .NET 8;
- собрал структуру MVVM без внешних библиотек;
- реализовал хранение данных в памяти на первом этапе;
- сделал главный экран: список операций, форма добавления, сводка по доходам, расходам и балансу;
- оформил документы и диаграммы в папке `docs`, добавил изображения для просмотра на GitHub.

## Как запустить
1. Открой `FinanceBuddy.sln` в Visual Studio 2022 на Windows.
2. Проверь, что установлен workload .NET Desktop Development.
3. Запусти проект `FinanceBuddy` через F5.
