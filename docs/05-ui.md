# Интерфейс приложения

## Макет главного окна
![Главное окно](img/ui-mainwindow.png)

## Главное окно
- слева таблица операций: дата, тип, категория, описание, сумма;
- снизу кнопки: удалить, редактировать, категории, обновить;
- справа форма добавления: тип, категория, дата, описание, сумма;
- сверху сводка: доход, расход, баланс.

## Окна редактирования
- **Редактирование операции** — изменяет поля выбранной операции и сохраняет в БД.
- **Категории** — добавление/удаление категорий.
