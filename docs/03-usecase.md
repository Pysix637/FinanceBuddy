## Смысл диаграммы
Основные сценарии пользователя:
- добавить операцию (доход/расход);
- редактировать операцию;
- удалить операцию;
- просмотреть список операций;
- управлять категориями;
- посмотреть сводку (доход, расход, баланс).

Сводка пересчитывается при изменении набора операций.
