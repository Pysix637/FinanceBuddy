# Порядок разработки

## Этапы, которые я выполнил
1. Сформулировал задачу и требования.
2. Создал каркас WPF-проекта на .NET 8.
3. Настроил MVVM (CommunityToolkit.Mvvm) и внедрение зависимостей через Host.
4. Реализовал `MainViewModel` и разметку главного окна.
5. Реализовал сервисы для ViewModel (парсер суммы, расчёт итогов, сервис окон).
6. Реализовал слой Data:
   - EF Core + SQLite;
   - автоматическое создание базы;
   - тестовые данные (seed) при первом запуске.
7. Добавил окна редактирования:
   - редактирование операции;
   - управление категориями.
8. Оформил документы и диаграммы в `docs/` и добавил изображения в `docs/img/`.

## Проверка результата
- Запуск приложения и наличие тестовых данных.
- Добавление, редактирование и удаление операций.
- Добавление и удаление категорий (удаление запрещено, если категория используется).
