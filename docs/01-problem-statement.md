# Постановка задачи

## Тема проекта
**FinanceBuddy** — WPF‑приложение, персональный финансовый дневник для учёта доходов и расходов.

## Цель
Я делаю настольное приложение на WPF, чтобы пользователь мог:
- добавлять операции доход или расход с датой, категорией и описанием;
- просматривать список операций;
- видеть сводные показатели: доходы, расходы, баланс;
- на следующих этапах фильтровать операции, хранить данные в SQLite, экспортировать и импортировать JSON.

## Границы первого этапа
На первом этапе я реализовал каркас проекта и базовый экран, а данные храню в памяти.

## Ожидаемый результат
- репозиторий проекта на GitHub с исходниками;
- комплект документов по проекту в папке `docs`.
