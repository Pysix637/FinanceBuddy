# Инструменты разработки

## Платформа и язык
- C#, .NET 8
- WPF (Windows Presentation Foundation)

## Среда разработки
- Visual Studio 2022 (workload **.NET Desktop Development**)

## Пакеты NuGet
- `CommunityToolkit.Mvvm` — команды и биндинги для MVVM (ObservableObject, RelayCommand, Source Generators).
- `Microsoft.Extensions.Hosting` — запуск приложения через Host гарантирует единый способ регистрации сервисов.
- `Microsoft.Extensions.DependencyInjection` — DI контейнер.
- `Microsoft.EntityFrameworkCore` — ORM слой для работы с данными.
- `Microsoft.EntityFrameworkCore.Sqlite` — провайдер SQLite для EF Core.

## Инструменты для диаграмм
- PlantUML (код `.puml`) + экспорт в `.png` для просмотра на GitHub.
